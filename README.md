# Авто-установщик NovaDayz Shop

Официальный скрипт для автоматической установки, настройки и развертывания нашего донат-магазина NovaDayZ Shop на вашем сервере под управлением операционной системы **Ubuntu 20.04 / 22.04 / 24.04 LTS**.

## Безопасность и архитектура
Скрипт полностью следует лучшим стандартам системного администрирования Linux:
1. Создаёт отдельного, непривилегированного системного пользователя `novadayz`.
2. Запускает все процессы Node.js/Next.js и PM2 строго от лица этого пользователя, исключая работу из-под `root`.
3. Настраивает точечные права `sudo` без пароля для управления веб-сервером Nginx и автоматического выпуска SSL-сертификатов.
4. База данных MySQL создается с уникальным случайно сгенерированным паролем.
5. Исходный код вашего магазина скачивается по защищенному каналу с использованием SSH Deploy Keys.

---

## Требования для установки

Перед началом установки убедитесь, что:
1. У вас установлен чистый сервер **Ubuntu** (доступ по SSH под пользователем `root`).
2. Домен вашего магазина (например, `novadayz-shop.ru`) уже направлен на IP-адрес вашего сервера (A-запись в DNS).
3. У вас есть ваш **Steam Web API Key** (получить его можно на [Steam Developer](https://steamcommunity.com/dev/apikey)).

---

## Быстрая установка в 1 команду

Подключитесь к вашему чистому серверу по SSH (как `root`) и запустите команду:

```bash
curl -sSL https://raw.githubusercontent.com/Behemiron/InstallNovaDayzStore/main/install.sh | sudo bash
```

---

## Инструкция по интерактивным шагам:

1. **Домен**: Введите ваш домен без http/https (например, `novadayz-shop.ru`). Если домена нет, нажмите ENTER (сайт будет работать по IP-адресу).
2. **Steam API Key**: Вставьте ваш API-ключ Steam.
3. **DayZ API Key**: Введите секретный ключ для связи сайта с вашим игровым сервером DayZ (любая сложная строка).
4. **Репозиторий GitHub**: Нажмите ENTER (по умолчанию будет выбран наш приватный репозиторий `Behemiron/NovaDayzStore`).
5. **Настройка SSH Deploy Key**:
   * Скрипт автоматически создаст SSH-ключ на сервере и выведет его публичную часть на экран.
   * Скопируйте эту строку полностью.
   * Зайдите на GitHub в настройки приватного репозитория: **Settings ➔ Deploy keys ➔ Add deploy key**.
   * Вставьте ключ, дайте ему название (например, `VPS Deploy Key`) и нажмите **Add key** (права на запись Allow write access ставить **не нужно**).
   * Вернитесь в консоль сервера и нажмите **ENTER**.

Скрипт автоматически скачает файлы репозитория, соберёт бэкенд, фронтенд, настроит Nginx и выпустит SSL-сертификат Let's Encrypt для вашего домена.

После установки сайт полностью готов к работе!
