# NovaDayz Shop - Auto-Installer

Скрипт для быстрой автоматической установки и настройки донат-магазина NovaDayZ Shop на операционной системе **Ubuntu 20.04 / 22.04 / 24.04 LTS**.

## Что делает установщик?
1. Обновляет пакетную базу Ubuntu.
2. Устанавливает: **Node.js 20 LTS**, **Git**, **PM2**, **Nginx**, **MySQL-Server**, **Certbot**.
3. Создаёт базу данных MySQL, генерируя случайные надежные пароли.
4. Генерирует SSH Deploy Key для безопасного скачивания файлов из вашего приватного репозитория.
5. Скачивает и собирает бэкенд и фронтенд.
6. Настраивает виртуальный хост Nginx и выпускает SSL-сертификат Let's Encrypt (если указан домен).
7. Запускает все процессы под управлением PM2 с автоматическим запуском после перезагрузки сервера.

---

## Подготовка к установке

Перед запуском установщика убедитесь, что у вас есть:
1. Чистый сервер с ОС **Ubuntu** и доступом пользователя `root`.
2. Направленный на IP вашего сервера домен (опционально, для работы HTTPS).
3. **Steam Web API Key** (можно получить бесплатно на [Steam Community](https://steamcommunity.com/dev/apikey)).
4. Название вашего приватного репозитория на GitHub (например, `Behemiron/NovaDayzStore`).

---

## Запуск установки в 1 клик

Подключитесь к вашему новому серверу по SSH под пользователем `root` и выполните следующую команду:

```bash
curl -sSL https://raw.githubusercontent.com/Behemiron/InstallNovaDayzStore/main/install.sh | sudo bash
```

*(Замените плейсхолдеры в URL на имя вашего публичного репозитория, куда вы поместите файл `install.sh`)*.

---

## Шаги интерактивной установки:

1. **Домен**: Введите доменное имя (например, `mydayzshop.ru`) или нажмите Enter, чтобы использовать IP сервера.
2. **Steam API Key**: Вставьте ваш API-ключ Steam.
3. **DayZ API Key**: Придумайте секретный ключ для связи сайта с игровым сервером (например, случайный сложный пароль).
4. **Репозиторий GitHub**: Введите имя вашего приватного репозитория (например, `Behemiron/NovaDayzStore`).
5. **SSH Deploy Key**: 
   * Установщик сгенерирует ключ и выведет его на экран.
   * Скопируйте этот ключ.
   * Зайдите на GitHub в настройки вашего приватного репозитория: **Settings ➔ Deploy keys ➔ Add deploy key**.
   * Вставьте ключ, дайте любое название и нажмите **Add key**.
   * Вернитесь в консоль сервера и нажмите **ENTER**.

Установка продолжится автоматически. По завершении вы получите ссылку на сайт и пароль к вашей базе данных.
